# BlogGator-Go

SQLC isnt' an ORM its' more of a Go program that generates Go code from SQL Queries

Goose to manage our database Migrations (Schema)// go install github.com/pressly/goose/v3/cmd/goose@latest
SQLC(Generates go code that our application can use to interact with the database)// go install github.com/sqlc-dev/sqlc/cmd/sqlc@latest
Postgres driver so our program knows how to talk to our database // go get github.com/lib/pq
